# Global Malnutrition Trends: A Power BI Analysis (1983–2019)

## Overview
This project analyzes global under-five malnutrition indicators using Power BI.
The dashboard covers stunting, wasting, underweight, and overweight trends
across countries and income classifications from 1983 to 2019.

## Objectives
- Analyze country-wise malnutrition indicators
- Compare income classifications
- Identify high-risk regions
- Visualize trends over time

## Tools Used
- Power BI Desktop
- DAX
- CSV Dataset

## Files Included
- `/dashboard` – Power BI (.pbix) file
- `/data` – Dataset used for analysis
- `/reports` – Project report (PDF)
- `/screenshots` – Dashboard images
